## 1. Дефицит кадров для эксплуатации АЭС

Потребность в специалистах на этапе эксплуатации АЭС составляет 2000 человек.

- 20% (400 человек) — узкопрофильные ядерщики (операторы реактора, специалисты по радиационной безопасности).
    
- 80% (1600 человек) — специалисты по IT, КИПиА, энергетике (программисты, автоматизаторы, электрики).
    

Текущий выпуск специалистов по вузам:

- КазНУ: 20–30 человек в год по ядерной физике и смежным направлениям.
    
- Satbayev University: 15–25 человек в год по энергетике и автоматизации.
    
- ЕНУ: 10–15 человек в год по физике и IT.
    
- ВКТУ: 20–30 человек в год по энергетике и КИПиА.
    

Институт ядерной физики в Алматы (исследовательский реактор ВВР-К) обеспечивает практику для 50–70 студентов ежегодно.

| Показатель              | Текущие возможности (чел/год) | Потребности (чел/год, эквивалент) |
| ----------------------- | ----------------------------- | --------------------------------- |
| Узкопрофильные ядерщики | 45–80                         | 400                               |
| IT, КИПиА, энергетики   | 65–100                        | 1600                              |
| **Итого**               | **110–180**                   | **2000**                          |
